## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

`Python 3` installed on your local machine.

### Installation

```sh
pip install -r requirements.txt
```

## Usage

### Run App

```sh
python main.py YYYYMMDD # YYYY: Year MM: Month DD: Day
```
